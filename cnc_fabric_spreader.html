Automatic Fabric Spreader

CNC Fabric Spreading Machines

The automatic fabric Spreader is used to spread the large garments in the machine and fetches the amount of time to align the garment and calibrate the pattern design of the garment.

Description


The CNC fabric Spreading Machine contains a cloth length setting memory device which is controlled by computer for acceleration and deceleration of the cloth machine. It has slotted automatic tracking device according to cloth width it sets cutter walking distance.
It also has tow bar, emergancy stop device, automatic lifting device, cloth layer counter, automatic align edge device, rewinding device, one side fixed type movable presser, mobile clamp device, double – pull with a fixed pressure device and cloth auto cutter device.
